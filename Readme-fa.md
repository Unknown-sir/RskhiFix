#RskhiFix

RskhiFix یک اسکریپت برای ساخت و مدیریت چند سرویس systemd است که با `ping` کردن IP لوکالِ سمت مقابل، ترافیک keepalive ایجاد می‌کند تا تونل خواب نماند.

نصب (روش پیشنهادی با curl)

```
bash <(curl -fsSL https://raw.githubusercontent.com/Unknown-sir/RskhiFix/main/install.sh)
```
نصب (روش wget)
```
wget -qO- https://raw.githubusercontent.com/USERNAME/REPO/main/install.sh | bash
```

بعد از نصب

پنل پروژه با دستور زیر اجرا می‌شود:

```
Rskhi
```
